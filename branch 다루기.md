### branch 관련 명령어
***

<h4>branch 만들기</h4>

```
git branch {브랜치명}
git branch develop
```

<h4>branch 이동하기 </h4>

```
git checkout {브랜치명}
git checkout develop
```

<h4>branch 만들면서 이동하기</h4>

```
git checkout -b develop
```

<h4>branch 검색</h4>

```
git branch
```

<h4>branch 삭제</h4>

```
git branch -d
```

<h4>branch 병합</h4>

```
git merge {합치려는 브랜치명}
```
> 현재 작업중인 브랜치와 병합됨

<h4>branch 병합 취소</h4>

```
git merge --abort
```
